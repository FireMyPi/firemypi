FireMyPi Release Notes

Version 1.1

Version: v1.1
Date: Wed Jun 26 23:17:45 2024 -0600
  1. Version 1.1 update of FireMyPi.
  2. To migrate to this version, save your ‘config’ and ‘secrets’ directories. Delete your current firemypi directory and clone the new version from github.
  3. Note that this release changes the format of some of the ‘config’ and ‘secrets’ files. If using your saved version 1.0 files, update them with the version 1.1 changes or re-create them with the ‘mk’ scripts.
