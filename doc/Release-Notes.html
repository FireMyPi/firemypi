FireMyPi Release Notes

Version 1.7

Version: v1.7
Date: Mon Dec 9 16:27:24 2024 -0700
  1. Add cdc-ncm driver for antec ethernet dongle.
  2. Move ‘turn on journaling’ from rcsysinit to firemypi-configure.
  3. To migrate to this version, ‘git pull’ the changes or save your ‘config’ and ‘secrets’ directories, delete your current firemypi directory and clone or download the new version from github.

Version 1.6

Version: v1.6
Date: Fri Oct 25 18:59:04 2024 -0600
  1. Turn on journaling for root in ‘fsresize’ instead of ‘partresize’. This fixes a problem that showed up with core 189 where the root file system would get corrupted when journaling was turned on.
  2. To migrate to this version, ‘git pull’ the changes or save your ‘config’ and ‘secrets’ directories, delete your current firemypi directory and clone or download the new version from github.
